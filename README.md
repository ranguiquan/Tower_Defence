# Tower_Defence
A tower defence game demo

### 现在有的实体
+ 塔
+ 敌人
+ 子弹
### 现在实现的功能
+ 鼠标左键点击菜单选择炮塔，左键在地图上放置，右键取消
+ 炮塔自动攻击一定范围内的敌人，子弹碰撞消失，敌人扣血
+ 炮塔向目标方向旋转，旋转就绪后才能开火
+ 出画面对象全部消灭
+ 敌人生成器
+ 增加敌人种类
